# Meu Portfólio — Single Page Application (SPA)

Este projeto é o meu portfólio profissional, desenvolvido como uma Single Page Application (SPA), com foco em performance, clareza e experiência do usuário. Ele foi planejado para apresentar minha trajetória, minhas habilidades e meus projetos de forma objetiva e otimizada. Ele também funciona é um projeto real de desenvolvimento, onde aplico as boas práticas e tecnologias que utilizo no meu dia a dia.  

O portfólio também funciona como um projeto real de desenvolvimento, onde aplico as boas práticas e tecnologias que utilizo no meu dia a dia.  

### Principais objetivos
- Criar uma interface rápida, responsiva e acessível
- Demonstrar minhas habilidades técnicas na prática
- Organizar meus projetos de forma clara e profissional
- Servir como um exemplo real da minha capacidade de criar interfaces escaláveis e bem estruturadas

## Tecnologias utilizadas
- Next.js para renderização eficiente, roteamento e SEO
- React para construção dos componentes reutilizáveis
- TailwindCSS para estilização ágil e consistente
- Framer Motion (se usar) para transições suaves
- Zod + RHF em formulários
- Google Analytics e Search Console para monitoramento e SEO
- reCAPTCHA para proteção do formulário de contato
- Resend para envio de mensagens via API (contato)

### Principais features
- Design responsivo (mobile-first).
- Seção “Sobre mim” profissional e detalhada.
- Stack organizada (frontend, backend, devops, estrutura, DevOps e etc.).
- Galeria de projetos com descrição técnica e links.
- Seção de contato funcional com validação e proteção.
- Otimização SEO básica para indexação adequada.

### O que este projeto demonstra
- Minha capacidade de planejar e desenvolver uma SPA completa.
- Uso real de ferramentas modernas do ecossistema JavaScript.
- Escrita limpa, componentização e boa organização do código.
- Atenção à experiência do usuário e boas práticas visuais.
- Preocupação com acessibilidade, segurança e performance.
